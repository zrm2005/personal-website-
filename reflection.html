Course Reflection

This course has helped me better understand how websites are built using HTML and CSS. At the start, I mostly thought about websites from a user perspective, but now I understand more about the structure behind them and the importance of organizing content clearly and logically.

One of the most useful things I have learned is how to create multiple connected pages with navigation. Building a site with a consistent layout across pages has shown me how important design consistency is for usability and professionalism. It also helped me understand how file organization and linking work in real projects.

Growth and Goals

I have also improved my ability to format content using headings, paragraphs, links, lists, and semantic elements. Learning how to separate structure from styling using external CSS files made the process much more organized. It showed me that even simple design choices can make a large difference in how polished a website looks.

Going forward, I want to continue improving my web development skills by learning more advanced CSS, JavaScript, and responsive design techniques. This class has given me a strong foundation and has made me more confident in my ability to build websites that are both functional and visually clean.
